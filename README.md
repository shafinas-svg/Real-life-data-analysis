# Real-life-data-analysis

## Описание проекта

Проект посвящён анализу данных о питании, тренировках, сне и физической активности во время похудения.

Данные основаны на реальном образе жизни и включают:

- тренировки в зале
- степ-аэробику
- питание
- калории
- белок
- сон
- шаги
- воду
- настроение
- уровень энергии

Главная цель проекта — понять, какие закономерности можно найти в данных и как режим влияет на самочувствие и прогресс.

---

## Цель проекта

С помощью Python, pandas и numpy проанализировать личные данные о:

- здоровом питании
- физической активности
- сне
- уровне энергии

и получить осмысленные выводы на основе анализа.

---

## Данные

Файл с данными:

```text
data/fitness_data.csv
```

В таблице содержится более 100 строк.

Каждая строка — это один день наблюдений.

### Основные столбцы

| Столбец | Описание |
|---|---|
| дата | дата наблюдения |
| день_недели | день недели |
| выходной | является ли день выходным |
| зал | была ли тренировка |
| тип_тренировки | тип тренировки |
| длительность_тренировки_мин | длительность тренировки |
| степ_аэробика | была ли степ-аэробика |
| сон_часы | количество часов сна |
| вода_л | количество воды |
| калории_ккал | калории за день |
| белок_г | белок за день |
| шаги | количество шагов |
| питание_вне_дома | питание вне дома |
| сладкое | наличие сладкого |
| настроение_балл | настроение |
| энергия_балл | энергия |
| цель | цель режима |

## Запуск проекта

### Базовая статистика

```bash
python3 main.py stats
```

Выводит:

- средние калории
- средний белок
- среднее количество шагов
- средний сон
- среднее количество воды

### Анализ тренировок

```bash
python3 main.py training
```

Сравнивает тренировочные дни и дни без тренировок.

Анализирует:

- шаги
- белок
- энергию

### Анализ сна

```bash
python3 main.py sleep
```

Сравнивает дни:

- со сном 7+ часов
- со сном меньше 7 часов

### Анализ питания

```bash
python3 main.py nutrition
```

Анализирует:

- выполнение нормы белка
- соблюдение калорий
- питание вне дома
- сладкое

### Основные выводы

```bash
python3 main.py report
```

Выводит готовые выводы по данным.

## Что реализовано

### Загрузка данных

Файл `table_operation.py`:

- загружает CSV
- преобразует дату
- приводит числовые столбцы к нужному типу

### Анализ данных

В `analysis.py` реализованы функции:

- `basic_stats`
- `analyze_training_days`
- `analyze_sleep`
- `analyze_nutrition`
- `get_conclusions`

### Декораторы

В `decorators.py` реализованы декораторы:

#### validate_dataframe

Проверяет:

- является ли объект DataFrame
- существуют ли нужные столбцы

#### round_numeric_results

Автоматически округляет результаты анализа.

### Генераторы

В `generators.py` реализованы генераторы:

- `iter_all_days`
- `iter_training_days`
- `iter_high_protein_days`

Они позволяют перебирать строки таблицы постепенно.

### CLI-интерфейс

Проект запускается через терминал с помощью argparse.

Примеры:

```bash
python3 main.py stats
python3 main.py training
python3 main.py sleep
python3 main.py nutrition
python3 main.py report
```
